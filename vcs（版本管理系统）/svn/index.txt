- 百灵K歌代码 版本管理迁移 eclipse 实现
  1. 创建trunk 三个文件
  2. 将整个项目 branch 到 trunk 文件夹内的新项目文件夹例如 larko-epg-h5.
  3. 删除trunk 文件夹以外的其他文件夹
  4. 创建 tags, branches文件夹
  5. 进入 eclipse import trunk 文件夹内的主线版本代码
  6. branch 分支 到branches/{{branchName}}
  7. 这个时候就可以 使用eclipse switch 来切换了
